# Project Example Skill

This is an example custom skill demonstrating the recommended structure.

## Description

Use this as a template for creating your own project-specific skills.

## Arguments

- `target`: The target to operate on (optional, defaults to "default")

## Steps

1. Analyze the current project state
2. Identify relevant files based on the target
3. Execute the appropriate action
4. Verify the results

## Usage

```
/project-example
/project-example target=specific-module
```

## Notes

- This skill is preserved during installer updates
- Modify or delete this example as needed
- Create new skills by copying this structure
